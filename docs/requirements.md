# 需求规格说明书 v3

## 1. 产品概述

### 1.1 产品名称
自媒体选题评价助手

### 1.2 产品目标
帮助自媒体创作者快速评估选题价值。用户粘贴选题描述，AI 自动从 9 个维度打分，给出综合评分和详细推理过程。

### 1.3 交互模式
- 用户在网页输入框中粘贴选题描述
- 点击「AI 自动评分」按钮
- 显示 4 步评分过程（理解选题 → 激活知识 → 逐维度评分 → 生成报告）
- 结果页展示：AI 对选题的理解 + 背景分析 + 各维度评分详情 + 加权计算过程
- 用户可保存评价到本地历史

---

## 2. 评价维度体系（9 维度 + 1 红线）

| # | 维度 | 权重 | 评分逻辑 |
|---|------|:----:|----------|
| 1 | 🔥 情绪的浓烈程度 | **30%** | 愤怒/不公/恐惧/猎奇/喜悦，情绪越浓烈越高分 |
| 2 | 🌀 猎奇程度 | **16%** | 颠覆常识/突破底线/反差荒诞，越出人意料越高分 |
| 3 | 🧠 默认认知程度 | **12%** | 一知半解=最高分；熟知=中等；一无所知=较低 |
| 4 | 🌐 共鸣广度 | **10%** | 个人→家庭→社会→国家→世界 |
| 5 | 🧩 内容晦涩程度 | **10%** | 故事越简单易懂越高分；逻辑复杂+硬核知识=越低分 |
| 6 | 📸 可视化潜力 | **8%** | 是否容易拍出好看/有趣的画面？画面感越强越高分 |
| 7 | 🤖 平台算法友好度 | **5%** | 完播/互动/分享预期如何。**UFO/外星人/超自然话题默认低分**（平台限流），除非是官方证实的重大事件 |
| 8 | 📡 流行热点关联度 | **5%** | 是否关联当下热点话题 |
| 9 | 👥 受众规模 | **4%** | 全民向/大众向/圈层向/小众向 |
| 10 | 💰 制作成本 | 参考 | 不计入总分，分数越高=成本越低 |
| 🚫 | 法律红线 | 否决 | 触碰→总分归零 |

> 权重合计：30+16+12+10+10+8+5+5+4 = **100%**

### 新增维度说明

**🧩 内容晦涩程度**：基于认知心理学「认知流畅性（Cognitive Fluency）」理论。信息越容易理解，越容易被接受和传播。
- 故事简单直白、行文轻松 → 高分
- 逻辑链复杂、多层反转、涉及高中以上理科知识 → 低分

**📸 可视化潜力**：选题能否转化为好看的画面直接影响完播率和制作执行难度。
- 场景丰富、对比强烈、有视觉冲击 → 高分
- 纯抽象讨论、没有画面支撑 → 低分

---

## 3. 底线惩罚规则

> 如果 **内容晦涩程度 < 50** 且 **可视化潜力 < 50** 同时满足，总分额外扣 **20%**。
>
> 原因：观众既看不懂又没画面看，大概率中途划走，情绪和猎奇再强也触达不到。这是自媒体内容的"死刑"条件。

## 4. 分数等级

| 分数 | 等级 | 建议 |
|------|------|------|
| 85-100 | ⭐⭐⭐⭐⭐ 强烈推荐 | 全力以赴 |
| 70-84 | ⭐⭐⭐⭐ 值得做 | 投入时间准备 |
| 55-69 | ⭐⭐⭐ 斟酌 | 有亮点但存在短板，建议优化后再评估 |
| 40-54 | ⭐⭐ 不太推荐 | 暂缓 |
| 0-39 | ⭐ 不推荐 | 换选题 |
| 红线触发 | 🚫 一票否决 | 直接放弃 |

---

## 4. 评分过程透明化

每次评分展示 4 步过程：
1. 🔍 理解选题核心
2. 🧠 激活背景知识
3. 📊 逐维度评分
4. 📝 生成分析报告

结果页额外展示：
- AI 对选题的理解（供用户验证理解是否正确）
- 背景信息分析（AI 对该领域的认知）
- 加权计算过程（公式可视化）
